# Prompts

This folder contains prompt templates and examples for using pdf-probe.

## Structure

- **examples/** — Template and example prompts (committed to version control, shared with community)
- **working/** — Developer's local working prompts (ignored by git, for personal use)

## Usage

1. Copy example prompts from `examples/` to get started
2. Develop and iterate on your prompts in `working/`
3. Share useful templates by moving them to `examples/` for the community Directory

Store PDF analysis prompts and templates here for pdf-probe.

This directory can contain:
- Sample analysis prompts for different PDF types
- Metadata extraction templates
- Custom extraction workflows
- Documentation prompts

## Usage

Reference prompts from this directory in your analysis workflows or documentation.
